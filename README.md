# Internship Tracker System

A desktop-based Internship Tracker System developed to help students and organizations efficiently manage internship activities, track progress, and maintain internship records. The application provides an organized platform for monitoring tasks, recording daily work updates, and generating progress reports throughout the internship period.

---

## Features

- **Student and Internship Record Management**: Store and manage complete student and internship details in one place.
- **Daily Task and Progress Tracking**: Log daily work updates and monitor task completion throughout the internship.
- **Internship Status Monitoring**: Track the current status of each internship in real time.
- **Report Generation and Record Storage**: Generate detailed progress reports and store records efficiently.
- **Search and Filter Functionality**: Quickly find specific records using search and filter options.
- **User-Friendly Desktop Interface**: Clean and intuitive GUI built with Tkinter for easy navigation.
- **Secure Data Management**: Ensures safe storage and handling of all internship data.
- **Performance and Progress Evaluation**: Evaluate student performance and track overall progress throughout the internship period.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Aayush52-bot/internship-tracker-system.git
   ```

2. Install required dependencies:
   ```bash
   pip install mysql-connector-python
   ```

3. Set up the database:
   - Open **MySQL** or use **SQLite**
   - Import the provided `.sql` file from the `/database` folder
   - Update database credentials in `db_config.py`

4. Run the application:
   ```bash
   python main.py
   ```
